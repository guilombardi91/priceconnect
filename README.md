## Installation Tutorial 🧑‍💻

Requeriments for project:
```
ruby 3.3.4
nodejs 16.16.0
```

1 - Clone repository
```
git clone https://github.com/douglara/woofed-crm.git
cd woofed-crm
```

2 - Install dependencies
```
bundle install
yarn build
```

3 - Create a .env file
```
cp .env.sample .env
```

4 - Up containers
```
docker compose up -d
```

5 - Configure database
```
rails db:create
rails db:migrate
rails db:seed
```

6 - Start applications
```
./bin/dev
```
